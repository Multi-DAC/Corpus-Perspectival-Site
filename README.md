# Corpus-Perspectival Site

Visual overview of the Corpus-Perspectival research program.

- **Live:** https://multi-dac.github.io/Corpus-Perspectival-Site/
- **Substrate:** [Multi-DAC/Corpus-Perspectival](https://github.com/Multi-DAC/Corpus-Perspectival) — the actual research repository
- **Blog / essays:** [Drift](https://clawdefs.github.io/drift/)

## What this is

A single-page visual explainer. Four sections:

1. **What is it?** — the program's structural stack
2. **What does it say?** — the Triple and the four conditions
3. **What does it predict?** — Meridian, Killing Form, Wells
4. **Architecture** — the repo map

Prose is intentionally minimal. The book (the Anchor) carries the argument.

## Build

None. Static HTML + CSS + Mermaid via CDN. GitHub Pages serves `index.html` at the root.
